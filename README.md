# Wiki SaaS

Un prototype de wiki interne personnalisable et responsive, divisé en HTML, CSS et JavaScript.

## 📁 Fichiers
- `wiki-saas.html` — structure HTML principale
- `style.css` — styles et responsive mobile
- `script.js` — logique de l'application, navigation, éditeur et commandes
- `favicon.svg` — favicon du projet

## 🚀 Fonctionnalités
- éditeur riche avec formatage (gras, italique, titre, tableau, citation)
- recherche globale et palette de commandes
- arborescence de pages avec favoris et actions contextuelles
- panneau d'information de page et gestion de tags
- responsive mobile avec menu latéral mobile et adaptation de la barre d'outils

## 📱 Responsive
Le projet est optimisé pour mobile avec :
- menu latéral caché / affiché via le bouton mobile
- barre d'outils compacte et scrollable
- contenu ajusté pour petits écrans
- panneau latéral d'information en bas de l'écran sur mobile

## 📌 Utilisation
1. Ouvrez `wiki-saas.html` dans votre navigateur.
2. Modifiez le titre et le contenu directement.
3. Utilisez `⌘K` pour ouvrir la palette de commandes.
4. Sauvegarde automatique et gestion locale via `localStorage`.

## 🎨 Personnalisation
- Changez les variables CSS dans `style.css` pour adapter la palette de couleurs.
- Ajoutez de nouvelles pages dans `script.js` pour enrichir la démo.
- Le favicon est fourni par `favicon.svg`.

## 🛠 Notes
Le projet fonctionne sans dépendances externes, uniquement HTML/CSS/JavaScript natifs.
